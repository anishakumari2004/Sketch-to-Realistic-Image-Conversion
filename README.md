This project focuses on developing an AI-based system that converts hand-drawn sketches into realistic images using advanced deep learning techniques. It combines image preprocessing, feature extraction, and generative modeling to understand the structure of a sketch and enhance it with detailed textures, colors, and lighting. The system uses a Convolutional Neural Network, specifically ResNet-18, to extract meaningful features from the sketch, and a diffusion model with cross-attention to generate high-quality images. The main objective of this project is to bridge the gap between rough visual ideas and realistic outputs, making the design and visualization process faster, more efficient, and accessible across various domains.
